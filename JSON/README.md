# JSON(ジェイソン)とは？
## JSONの基本情報
- ファイルの拡張子は".json"
- JSONはいくつかの情報を持ったオブジェクトを表す
- 各プログラミング言語にはJSONを扱うためのライブラリが用意されている事が多い

こんな感じのやつ↓
```
{
	“name”: “リザードン”,
	“weight”: 90.5,
	“height”: 1.7,
	“Type”: [ “ほのお”, “ひこう” ],
	“character”: “もうか”,
        "male": true
}
```
